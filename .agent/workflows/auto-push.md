description: Automatically push all changes to GitHub after any code edit
---

After making any code changes to the masala-box project, always run the following steps:

// turbo-all

1. Stage all changes:
```bash
git add -A
```

2. Commit with a descriptive message summarizing the changes:
```bash
git commit -m "<descriptive message>"
```

3. Push to origin main:
```bash
git push origin main
```
